AsyncNinja: Simple and Safe Concurrency for Swift
download .ZIPdownload .TGZ

Provides typesafe, threadsafe, memory leaks safe concurrency primitives for Swift 3.

Includes Primitives:

  • Future (also known as promise or defer) is a proxy of value that will be available at some point in the future
  • Channel (also known as stream or observable) is a proxy of values that become available periodically
  • FiniteChannel is a combination of Future and Channel
  • Executor is object made to encapsulate a way of async execution
  • ExecutionContext is a protocol concurrency-aware objects must conform to. It basically make them actors or components of actor
  • Fallible is validation monad. Is an object that represents either success value of failure value (Error).

Supports: iOS, macOS, watchOS, tvOS. Linux support is in near future.
